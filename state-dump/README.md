# `dispute-state-dump`

This folder contains a script that generates the setup state for the `Arbiter` simulation's EVM.

## Regenerating Dumpfile

Dependencies:

- [`forge`][foundry]
- [`jq`][jq]

```sh
./dump.sh
```

[foundry]: https://github.com/foundry-rs/foundry
[jq]: https://github.com/jqlang/jq
